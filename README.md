# 常见问题记录

## 输入地点后的1小时到达圈是个很小的六边形？
高德api问题，可以尝试将地点改为附近的位置再试。

## 出行方式改变后，1小时到达圈阴影不变？
高德api问题，目前只有公交到达圈api,此参数实测无效。

## 导入csv文件后无反应？
编码问题，尝试使用notopad++打开csv文件，若为GB2312,则菜单--格式--转为utf-8编码格式。

## 点击一个地址后，无法规划路线？
可能是网络问题，尝试换一个其他点再试。
